# Gear VR Contoller 사용법

- 영상 목록 화면
1. 터치패드 상/하 : 선택 영상 변경
2. trigger : 선택 영상 시청
3. volume : 볼륨변경
- 비디오 플레이어 화면
1. 터치패드 좌/우 : 5초 뒤/앞으로
2. trigger : 재생메뉴 토글
3. back : 영상목록 화면으로 이동
4. home : orientation 변경
5. volume : 볼륨변경
